# Chess-bot
Bot for chess competition on Discord </br>
See more of the bot on [this link](https://mabule.github.io/chess.com/index.html)

The prefix for commands is "!"

Commands are :
*_help
*result
*result_confirm
*score_player
*score_class
*register
*modif_class
*modif_player
*admin
*list_player
*delete_class
*delete_player
*reset
*rebuild
*poule
*edit_color_profil
*show_profil
